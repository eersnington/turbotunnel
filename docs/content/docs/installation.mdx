Install Turbotunnel globally to use `tt` in any project. Contributors can run the same CLI from this repository with Bun.

## Requirements

Turbotunnel requires Node.js 22 or newer. Deploying a gateway also requires the Vercel CLI and a Vercel account that can create projects.

## Install globally

Install the CLI and Vercel CLI, then authenticate:

```sh
npm i -g turbotunnel vercel
vercel login
```

Both `tt` and `turbotunnel` run the same CLI:

```sh
tt --help
turbotunnel --help
```

## Run from the repository

The repository uses Bun 1.3.14 or newer:

```sh
bun install
bun run tt -- deploy
bun run tt -- http 5173
bun run tt -- dev --port 1024 -- bun run --cwd docs dev
```

The first `--` passes the remaining arguments through Bun's `tt` workspace script. For a custom `tt dev` command, the second `--` separates Turbotunnel options from the child command.
